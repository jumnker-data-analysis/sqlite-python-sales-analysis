# SQLite + Python Sales Analysis

Small beginner data project using:
- SQLite (database)
- Python (sqlite3 + pandas)
- Simple revenue analysis per product

## How to run
1) Create venv
python -m venv .venv

2) Activate
.\.venv\Scripts\Activate.ps1

3) Install dependencies
pip install -r requirements.txt

4) Run
python main.py
